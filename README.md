# 微信小程序商城首页 Demo

WeChat Mini Program — Shopping Mall Homepage.  
微信原生小程序商城首页模板。

## ✨ Features / 功能

- 🔍 **搜索栏** — 顶部商品搜索
- 🎠 **Banner 轮播** — 活动推广轮播图
- 🏷️ **分类导航** — 5 个商品分类快捷入口
- 🛍️ **商品网格** — 热门推荐双列瀑布流
- 🛒 **加入购物车** — 点击 + 号加购，TabBar 角标更新
- ⬇️ **触底加载** — 上拉加载更多

## 🛠️ Tech Stack / 技术栈

- 微信原生框架 (WXML + WXSS + JS)
- 微信小程序 API

## 📂 Project Structure / 项目结构

```
mini-shop/
├── app.js                  # 小程序入口
├── app.json                # 全局配置
├── app.wxss                # 全局样式
├── project.config.json     # 项目配置
└── pages/
    └── index/
        ├── index.wxml      # 首页模板
        ├── index.wxss      # 首页样式
        ├── index.js        # 首页逻辑
        └── index.json      # 页面配置
```

## 🔧 How to Run / 运行方式

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开工具 → 导入项目 → 选择 `mini-shop/` 目录
3. 使用测试号 AppID 或自己的 AppID
4. 点击编译预览
